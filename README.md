# Migration Toolkit - Installation Package

## What's Included

This package contains everything you need to run the Migration Toolkit on any Windows machine:

- **MigrationToolkit-Web** - Electron desktop application
- **VGMigrations** - PowerShell automation scripts
- **INSTALL.ps1** - Interactive installer

## Quick Start

### Prerequisites

- **Windows 10/11** (64-bit)
- **PowerShell 5.1+** (included with Windows)
- **Node.js 18+** (will prompt to install if missing)

### Installation Steps

1. **Copy this entire folder** to the target machine
2. **Right-click INSTALL.ps1** and select **"Run with PowerShell"**
3. **Choose installation location** (default: C:\MigrationToolkit)
4. **Wait for installation** (2-5 minutes)
5. **Launch from desktop** - Double-click "Migration Toolkit" icon

## First-Time Configuration

1. Click Settings icon (top right)
2. Configure API credentials
3. Add customers
4. Save settings
