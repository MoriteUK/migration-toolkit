Provision OneDrives


        

          
OneDrive Provisioning

          
Pre-provisions OneDrive sites for users in a mapping CSV. Requires Microsoft.Online.SharePoint.PowerShell.


          

            
            
          


          

            
            
          


          

            
            

              
              
            

            UPN column auto-detected from: Destination user, Destination, DestinationUPN, UserPrincipalName
          


          

            

              
              
            

            

              
            

          


          

            
            

              
              
            

          


          

            
            
          


          

            

          

        

      


      
      

        
Set Teams Owners


        

          
Teams Ownership Management

          
Adds a user as owner (and member if not already) to each Team/Group in the CSV. Requires Microsoft.Graph module.


          

            
            
          


          

            
            

              
              
            

            CSV must have an "address" column with team/group email addresses
          


          

            
          


          

            
          


          

            

          

        

      


      
      

        
Deduplicate Inventory


        

          
Discovery Workbook Deduplication

          
Classifies Teams, M365 Groups, and SharePoint sites from a discovery workbook, removing duplicates and group-connected sites. Outputs Discovery-Classified.xlsx next to the source file. Requires ImportExcel and Microsoft.Online.SharePoint.PowerShell modules.


          

            
            

              
              
            

            Must contain Teams, M365Groups, and SharePointSites worksheets
          


          

            
          


          

            

          

        

      


      
      

        
Purge Deleted SharePoint Sites


        

          
Permanently Remove Sites from Recycle Bin

          
Permanently purges destination SharePoint sites (from a mapping CSV) that are sitting in the SPO recycle bin. Run this before recreating sites if a previous test created and then deleted them. Requires Microsoft.Online.SharePoint.PowerShell module.


          

            
            

              
              
            

          


          

            
          


          

            

          

        

      


      
      

        
Get Domain Devices


        

          
Export Entra Registered Devices for a Domain

          
Connects to Microsoft Graph, retrieves all users whose UPN belongs to the specified domain, and exports their registered devices to a CSV. You will be prompted to sign in when the window opens.


          

            
          

        

      


      
      

        
Domain Removal Workflow


        

          
Complete Domain Removal Process

          
Guided 3-step workflow: updates on-premises UPNs, triggers Entra Connect sync, then removes the domain from M365. Collects all inputs in its own window.


          

            
1Update On-Premises UPNs

            
2Sync to Cloud (Entra Connect)

            
3Remove Domain from M365

          


          

            
          

        

      


      
      

        
Remove Domain


        

          
Remove M365 Domain Objects

          
Reads discovery CSVs and removes the corresponding M365 objects (mailboxes, groups, contacts, apps, devices, proxy addresses) via Exchange Online and Microsoft Graph. Select the discovery folder and objects to remove in the window.


          

            Warning: This permanently removes M365 objects. Ensure discovery CSVs are from the correct domain run.
          


          

            
          

        

      


      
      

        
Update On-Prem UPNs


        

          
Active Directory UPN Update

          
Reads CSVs from the selected folder, finds users with the source domain UPN, and updates their UPN, email, and proxy addresses in on-premises Active Directory.

Update Cloud UPNs


        

          
Microsoft 365 UPN Update

          
Connects to Microsoft Graph and updates cloud-only user UPN domain suffixes. On-premises synced users are skipped — update those in Active Directory instead. A device code sign-in prompt will appear in the log below.

Hide from Address Book


        

          
Exchange Global Address List

          
Sets HiddenFromAddressListsEnabled on Exchange Online recipients from discovery CSVs (mailboxes, distribution groups, mail contacts, shared mailboxes, M365 groups). A browser sign-in to Exchange Online will open when you click Run.

Remove Aliases


        

          
Remove Alias & IM Addresses

          
Strips selected address types from Exchange Online recipients in discovery CSVs. The primary SMTP: address is always preserved. Optionally filter by domain to target only addresses from a specific domain.
